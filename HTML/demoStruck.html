Dokumentasi untuk Programmer

      
Hal penting supaya struk bisa diprint dengan benar:

      
1) Load library print
<script src="assets/js/printCommon.js"></script>

2) Harus ada iframe sebagai print sink
<iframe id="printSink" class="print-sink"></iframe>

3) Payload print berupa daftar commandParts (string)
const printer = new PrintLib();
const commandParts = [
  "COMMANDS.TEXT_FORMAT.TXT_ALIGN_CT",
  PrintLib.wrap("BoostechLab Coffee\n")
];

4) Kirim ke iframe lalu panggil print()
printer.printParts(commandParts);

5) Urutan script demo
<script src="assets/js/demoStruck.js"></script>

      
Catatan: isi commandParts bisa disusun sesuai layout struk. Formatnya tetap mengikuti helper COMMANDS.

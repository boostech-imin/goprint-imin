Dokumentasi untuk Programmer

Hal penting supaya demo QR bisa dimuat dan diprint:

1) Load library print
<script src="assets/js/printCommon.js"></script>

2) Library QR wajib di-load sebelum demo QR
<script src="https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js"></script>

3) Harus ada iframe sebagai print sink
<iframe id="printSink" class="print-sink"></iframe>

4) Payload print berisi COMMANDS.QR()
const printer = new PrintLib();
const commandParts = [
  "COMMANDS.TEXT_FORMAT.TXT_ALIGN_CT",
  `COMMANDS.QR("${currentCode}", { align: "CENTER", moduleSize: 6, margin: 4, ecc: "M" })`
];

5) Kirim ke iframe lalu panggil print()
printer.printParts(commandParts);

6) Urutan script demo
<script src="assets/js/demoQR.js"></script>

Catatan: QR di browser hanya preview. Printer menggunakan perintah COMMANDS.QR() saat print.
